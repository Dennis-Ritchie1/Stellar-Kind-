# Scripts

This folder contains helper scripts and automation tools for the Stellar-Kind project.

## Recommended files

- `build.sh` — build-related helpers for frontend/backend/contract
- `deploy.sh` — deployment utilities for staging and production
- `migrate.sh` — database migration helpers
- `local-dev.sh` — commands to start local services and dependencies

## Purpose

Use this directory for scripts that simplify repetitive tasks, such as:

- project setup
- local development orchestration
- contract deployment helpers
- database migration automation
- environment validation
